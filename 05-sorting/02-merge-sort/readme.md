# Merge Sort

## Example
```
        [38 27 43 3]
        /         \
    [38 27]      [43 3]       Splitting
      /             \
  [38] [27]       [43] [3]
----------------------------
      \             /
    [27 38]      [3 43]       Merging 
        \          /
        [3 27 38 43]
```

## Runtime
`O(n log n)`
